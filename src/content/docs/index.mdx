This documentation site is the working home for New Jersey Innovation Authority brand design
guidelines. It is structured so contributors can add standards as decisions are approved, keep
source files close to usage guidance, and make public-facing work easier to review.

## Working Model

These pages should document approved guidance, not private drafts. When a standard is provisional,
label it clearly and include the decision needed before teams use it in production.

## Intended Audience

- NJIA staff creating public-facing documents, presentations, web pages, and outreach materials.
- Partner teams using NJIA marks or referencing NJIA programs.
- Designers, developers, writers, and vendors producing materials on NJIA's behalf.
