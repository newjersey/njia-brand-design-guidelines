Typography should help readers scan quickly, understand hierarchy, and trust the information in
front of them.

## Current Web Default

This site uses system sans-serif fonts through Starlight's default stack, customized to prefer
Inter when available. This keeps the documentation lightweight while final type decisions are still
being documented.

## Hierarchy

- Use one clear page title per page or artifact.
- Keep section headings short and descriptive.
- Avoid all-caps headings except for established labels.
- Use body copy sizes that remain readable on mobile screens and in printed handouts.

## Needed Decisions

Add final font families, licensing notes, fallback stacks, print specifications, and examples once
the typography system is approved.
