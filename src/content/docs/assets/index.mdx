Use this page to document where approved NJIA brand assets live and how teams should request or
contribute updates.

## Asset Types

- Logo files.
- Color tokens and swatches.
- Presentation templates.
- Document templates.
- Social and web graphics.
- Photography and illustration guidance.

## Versioning

Approved assets should be versioned and dated. Avoid replacing a source file in place without a
recovery path, especially when external partners may already be using a previous file.

## Publishing Rules

Only publish assets that are approved for reuse. Drafts, explorations, and vendor-only source files
should be clearly separated from public guidance.
