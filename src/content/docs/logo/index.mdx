Use this page to document approved NJIA logo files, spacing rules, minimum sizes, and incorrect
usage examples.

## Required Guidance

- Primary logo files for light and dark backgrounds.
- One-color logo files for constrained production contexts.
- Clear space measured from a stable element in the mark.
- Minimum sizes for print and digital use.
- Co-branding rules for partner materials.

## Usage Rules

Do not stretch, recolor, outline, rotate, crop, or rebuild the logo. Use approved source files and
keep enough surrounding space for the mark to remain legible.

## Needed Decisions

Add final logo files and exact spacing measurements once the approved identity assets are available.
